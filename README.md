# ريكسد — لعبة أسئلة الفرق 🎮

لعبه ريكسده عربية للفرق: اختر 6 فئات، وكل فئة فيها ثلاث درجات صعوبة
(200 سهل / 400 متوسط / 600 صعب جداً) مع مساعدات: إظهار خيارات، نقاط مضاعفة، وتغيير السؤال.

## المميزات
- +25 فئة: أسئلة عامة، دين، تاريخ، جغرافيا، علوم، رياضة، أنمي (ون بيس، ناروتو، سولو ليفلينق، هاجيمي نو إيبو)، بريكن باد، ريزدنت إيفل، فالورانت وغيرها.
- فئات مصوّرة بالكامل: أعلام الدول 🌐، جوازات السفر 🛂، شعارات العلامات، شعارات الألعاب — كل الصور متحقَّق منها.
- أسئلة «من في الصورة؟» بصور الشخصيات الرسمية.
- نظام عدم تكرار: السؤال لا يعود إلا بعد نفاد كل أسئلة مستواه (محفوظ مشفَّراً AES-GCM).
- نقاط مضاعفة: سؤال 200 → 400، سؤال 400 → 800، سؤال 600 → 1200.
- يعمل بالكامل من ملف واحد بدون خادم.
